# Documentation
This tool automates package management using Node.js

## Installation

To run this app you should follow such steps:
1. Install packages by `npm install`
2. Create .env file with following contents:
```
PRIVATE_KEY="-----BEGIN RSA PRIVATE\n... your private key for GithubApp goes here"
GITHUB_APP_IDENTIFIER=your app id
GITHUB_WEBHOOK_SECRET=your webhook secret
```
